O começo de tudo

Nasceu em 1989 pelo Holandês Guido van Rossum no Instituto de Pesquisa Nacional para Matemática e Ciência da Computação (CWI) para ser a linguagem de script de um novo sistema operacional distribuído chamado Amoeba. Em 1991 Guido anexou o código fonte da linguagem em um grupo de discussão chamado alt.sources, isso mostra que desde sua primeira versão já estava presente o espírito colaborativo que temos ainda hoje.

Primeiro programa

Python é uma linguagem multiplataforma nativa em Linux/Unix, e facilmente instalada em Windows. Não é necessário uma IDE para desenvolvimento, um bom editor de textos com sublime-text em conjunto com alguns plugins são suficientes para desenvolver bons softwares. Para fins de teste, o uso do próprio interpretador, sem a utilização de um editor de textos é indispensável, para usuários OSX existe um aplicativo chamado IPython como interpretador

Tipos de dados

Tudo em python é objeto, e sua filosofia vai além de outras linguagens de programação OO, todos os valores de dados no Python são encapsulados em classes de objetos, e tudo em um programa python é um objeto acessível a partir do programa.
